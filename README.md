# Wger Exercise Browser

A simple React Native app with Expo that lists exercises from the public Wger API. It is web-ready and works on mobile with the Expo toolchain.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start Expo:
   ```bash
   npm run web
   ```
3. Open the local web URL shown in the terminal.

## Features

- Fetches exercises from `https://wger.de/api/v2/exercise/?language=2`
- Displays a searchable, scrollable list
- Responsive card design for a clean web experience
- Simple React Native + Expo setup
